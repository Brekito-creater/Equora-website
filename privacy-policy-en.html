Privacy Policy

Last updated: April 12, 2026

1. General Information

Equora Wellness (hereinafter "the Application") is developed and operated by Brayner Eduardo Conrado Duque, an individual based in Colombia (hereinafter "the Developer").

The Application is designed for personal mental and emotional wellness tracking. The Developer is committed to protecting user privacy and ensuring the security of personal data.

For any questions regarding this Policy, contact us at: Equorapp@gmail.com

2. Data Collected

2.1 Registration data

Full name, age, email address, and a security PIN (stored as SHA-256 hash; Developer has no access to original PIN).

2.2 Daily tracking data

  • Mood
  • Anxiety level
  • Irritability level
  • Energy level
  • Concentration level
  • Sleep hours and quality
  • Physical activity

2.3 Health data

  • Medications: name, dosage, schedule, reminders
  • Emergency contact: name and phone number

2.4 Premium features

  • Biometric data (fingerprint/Face ID) for authentication only — processed by OS, never transmitted.
  • Custom tracking variables defined by the user.

3. Data Storage and Processing

All data is stored exclusively and locally on the user's device. The Application has no servers, does not transmit data over the Internet, and uses no third-party analytics or advertising.

  • The Developer has no access to user data at any time.
  • Data is not synced to the cloud.
  • Uninstalling the Application permanently deletes all data.
  • The user is solely responsible for backups.

4. Data Security

  • PIN stored as SHA-256 hash.
  • Optional biometric authentication (premium).
  • Data in device private storage, inaccessible to other apps.

No security system is completely infallible. The Developer cannot guarantee absolute security against unauthorized physical device access.

5. Data Sharing with Third Parties

The Application does not sell, transfer, or share personal data with third parties under any circumstances.

The only third-party components are Capacitor libraries (Ionic) for local native device functionality. They do not collect or transmit user data.

6. Minimum Age Requirement

The Application is intended for users aged 16 and older. We do not knowingly collect information from individuals under 16.

If you are a parent or legal guardian and become aware that a minor in your care has used the Application, please note that all data is stored exclusively on the device and the Developer has no access to it. The deletion of such information is the responsibility of the parent or legal guardian who has custody of the device. To delete all data, simply uninstall the Application or use the deletion option in settings. To report the case: Equorapp@gmail.com.

7. User Rights

  • Access their own data within the Application.
  • Correct inaccurate data through settings.
  • Delete all data by uninstalling the Application.
  • Export information in PDF via the reporting feature.

Since all data is local and the Developer has no access, these rights are exercised directly on the user's device.

8. Data Export

The Application allows export of wellness records in PDF. The file is generated locally and the user is responsible for managing it once exported.

9. Changes to This Privacy Policy

The Developer reserves the right to update this Policy at any time. Significant changes will be notified via an in-app notice. The current version is always available at equorawellness.com.
